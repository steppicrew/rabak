==index: Welcome to Rabak

##h1 Welcome to Rabak, your new backup tool!

##h2 What's Rabak?

Rabak is a reliable rsync based backup system, simple to configure,
simple to run, simple to restore data with.

Rabak's strategy is to always make full backups, usually to one or more
attached external hard drives. Using rsync's clever hard linking abilities,
the resulting backups use up little target space. Rabak uses simple and
flexible configuration files and supports backup to or from remote machines.
It even supports remote to remote backup (think remote control).

Rabak is very good for unattended safe remote backups, and has support
for postgresql and mysql backups via database dump streaming.

It's simple to configure different, multiple sources for one backup set.
There's support for file filters that hide the complexity of the rsync filters
(and make them actually usable).


==requirements: Rabak's system requirements

##h1 Requirements

You need Linux or Cygwin (Notice: On Cygwin pathes may not exceed 260 characters).
Other plattforms may work too. We didn't try.

You need rsync. I know rabak does't work before version 2.6.4.
Maybe even a newer one is requires. The newer the better.

You need Perl, I don't know which version. Probably >= 5.8

You need the *Mail::Send*, *Digest::SHA1*, *Getopt::Long*(>= 2.36), and *IPC::Run*
perl modules. Fetch it from CPAN:

##code
cpan -i Mail::Send (or cpan -i MailTools)
cpan -i IPC::Run
cpan -i Digest::SHA1
cpan -i Getopt::Long
##end

To use dupemerge, you will need the Perl module *Digest::SHA1*.
To install it do:
##code
cpan -i Digest::SHA1
##end


==installation: Install Rabak

##h1 Installation

Extract the files of rabak's archive to a temporary directory and
install Rabak's perl Library.
##code
cd rabak-<version>
perl Makefile.PL
make
make install
##end

Now you may copy the rabak script itself into your execution path
and dstributed config files to @/etc@.
##code
cp -p bin/rabak /usr/local/bin/
mkdir /etc/rabak
cp -p etc/* /etc/rabak/
##end


==quickstart: Quick Start

##h1 Quick Start

Look at rabak's help:
##code
rabak help
##end

Make a test backup:
##code
rabak backup -c rabak.sample.cf sample
##end

Have a look at the result:
##code
ls -lA test-bak
##end

Read the documentation:
##code
perldoc RabakLib::Doc
##end

Configure Rabak:
##code
cp /etc/rabak/rabak.sample.cf /etc/rabak/rabak.cf
nano -w /etc/rabak/rabak.cf
##end


==featuretests: Feature Tests

##h1 Feature Tests

If you have the right to run @mount@ and @mknod@, you can test more
features by doing:
##code
sh ./stuff/sample-env-make
##end

This produces a test environment in @/tmp/rabak-sample-data@. Data will be
backed up into @/tmp/rabak-sample-backup@. You can run the sample configurations
provided in @rabak.sample.cf@:
##code
rabak -c rabak.sample.cf backup sample_firstof
rabak -c rabak.sample.cf -i Tue backup sample_targetgroup
rabak -c rabak.sample.cf -i Tue2 backup sample_targetgroup
##end

When you're done, you can remove the test environment:
##code
sh ./stuff/sample-env-remove
##end

Check out the docs and the @*.cf@ files for more insight.

==sks: SKS-Segelschein

##h1 SKS-Segelschein 

Neben unseren Urlaubsfahrten bieten wir Ihnen auch die Möglichkeit eine SKS 
Prüfung abzulegen. Prinzipiell besteht diese Option für jede Woche, in welcher 
auch Urlaubsfahrten angeboten werden. Allerdings können die Angebote nicht 
gemischt genutzt werden.

Aktiv segeln und Segeln lernen steht ganz oben an, aber baden und schnorcheln in 
idyllischen Buchten oder ein Landgang in einem Hafenstädtchen sind reizvolle 
Ergänzungen, die Sie sich nicht entgehen lassen sollten.

Die Törns im Thyrrhenischen Meer verbinden die sportlichen Seiten des Segelns 
mit den atemberaubenden Eindrücken und der Vielfalt dieses Reviers. Von Elba aus 
segeln wir zu den malerischen Küsten Korsikas und Nordsardiniens.

Interessenten an dem SKS-Segelschein haben während des Törns genügend 
Gelegenheit, unter Anleitung des Skippers alle prüfungsrelevanten Manöver zu 
üben (etwas Segel-Vorerfahrung ist dennoch sinnvoll). Am letzten Törntag können 
Sie vor Elba die Praxisprüfung ablegen. 300 nm Segelerfahrung sind Voraussetzung 
für die Prüfungszulassung. Doch keine Bange, diese 300 nm ersegeln Sie sich in 
den Wochen vor der Prüfung. Dauer: 2 Wochen

- Kosten Nebensaison: 1.060 €/Person + Prüfungsgebühren vor Ort und Bordkasse
- Kosten Hauptsaison: 1.280 €/Person + Prüfungsgebühren vor Ort und Bordkasse

==skipper_training: Skippertraining

##h1 Ab April 2006 Skippertraining auf dem Kreuzerkatamaran Maya
; ##h2 Für Gruppen ab 3 Personen. Nach Vereinbarung

Die meisten Segler haben Erfahrung fast ausschliesslich mit Jollen und Yachten 
gemacht, allesamt Einrümpfer. Die Skipper wissen die modernen Monos gut zu Händeln,
Einen Katamaran jedoch haben nur vergleichsweise wenige geführt.

Ein Kat ist auch nur ein Segler und unterliegt denselben Bedingungen von 
Wind und Wellen, allerdings mit 2 Rümpfen, die getrennt voneinander, wenn auch 
miteinander verbunden, durchs Wasser pflügen. Diese Tatsache erfordert in vielen 
Bereichen ein Umdenken, bei den Manövern wie beim Segeln und bei der Sicherheit.

Wir trainieren alles was Sie auf dem Mono schon gelernt haben, ob Sie viel oder 
erst wenig Erfahrung sammeln konnten. Sie sind Skipper für einen Tag. Allerdings 
mit Netz und doppeltem Boden. Sie merken bald, auf einem Kat ist alles ein wenig 
anders, nicht nur der unschlagbare Komfort.

{bold Im Einzelnen vermitteln und üben wir:}
- An- und Ablegen vom Liegeplatz
- Segelsetzen, -reffen und -bergen
- Wenden und Halsen
- Der richtige Trimm
- Beiliegen
- Mann über Bord - Manöver
- Ankern
- Umgang mit Schiffsmotoren
- Umgang mit der elektrischen Einrichtung
- Verhalten bei Starkwind und Sturm
- Verwendung von Radar

Schwerpunkte setzen jeweils die einzelnen Teilnehmer.
Eine abschliessende Prüfung findet {bold nicht} statt.

- Kosten Nebensaison: 530 €/Person/Wo + Bordkasse
- Kosten Hauptsaison: 640 €/Person/Wo + Bordkasse
